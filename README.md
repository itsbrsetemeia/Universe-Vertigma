🌌 Vertigma

"Todo universo possui heróis. Todo multiverso possui lendas. Vertigma possui ambos... e algo muito além."

Bem-vindo ao Vertigma, um universo original de super-heróis, ficção científica, fantasia, horror cósmico e aventura, criado para contar histórias que ultrapassam os limites de um único mundo.

Aqui, tecnologia, magia, ciência, entidades ancestrais, civilizações alienígenas e forças cósmicas coexistem em um multiverso repleto de conflitos, mistérios e personagens memoráveis.

📖 Sobre

Vertigma é um projeto independente de construção de universo (Worldbuilding) inspirado pela grandiosidade de franquias como Marvel, DC, Invincible, The Boys, One Piece, Star Wars e muitas outras, mas com identidade própria.

O objetivo é criar um universo vivo, onde cada personagem, organização, planeta e evento possui propósito dentro de uma história maior.

📚 Lore

Cada arquivo deste repositório representa uma pequena parte do universo.

Nenhuma informação existe isoladamente: personagens influenciam eventos, eventos moldam civilizações e civilizações alteram o destino do multiverso.
